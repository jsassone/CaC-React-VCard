El proyecto combina todas sus funcionalidades (html, css y javascript) en un sólo archivo debido a su diminuto tamaño, para mantener la practicidad en la edición.
No posee comentarios relevantes sobre su funcionamiento porque el código se explica por sí mismo gracias a su sencillez y simplicidad.
La imagen del avatar se encuentra en una carpeta aislada por una cuestión de orden.

Si el proyecto escalara a un mayor tamaño, todos los distintos formatos de archivos (incluyendo los diferentes lenguajes, bbdd, etc) serían separados en carpetas organizando un directorio. Además el código estaría comentado para facilitar su comprensión.
